* 连接服务器

  ```commonlisp
  ssh user@.....
  ```

* 创建文件

  ```
  touch filename
  ```

* 创建文件夹

  ```commonlisp
  mkdir name
  ```

* 杀死进程

  ```
  kill -9 process_id
  ```

  ```
  pwd
  ```
